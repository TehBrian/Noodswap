# Frames

Place custom transparent frame images here.

- File name format: `<frame_key>.png` (or `.webp`)
- Supported frame keys: `buttery`, `gilded`, `drizzled`
- Bundled frames in this repo currently include all three keys.
- `gilded` and `drizzled` are temporary placeholders (currently duplicated from `buttery`) until distinct art is added.
- Recommended canvas: `350x490` RGBA
- Safe area: keep important details near edges; center should remain transparent for card art visibility.
