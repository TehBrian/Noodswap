# Card Fonts

Optional custom font overrides for card text rendering.

- File name format: `<font_key>.ttf`
- Optional bold file: `<font_key>-bold.ttf`
- Supported font keys:
  - `serif`
  - `mono`
  - `script`
  - `spooky`
  - `pixel`
  - `playful`
- `classic` is the built-in default style and is not a modifier key.
